## Installation

This plugin is a ZXP extension and can be installed using a ZXP installer.

---

### Step 1 — Download a ZXP Installer

Download and install a ZXP installer (e.g. AEScripts ZXP Installer):

https://aescripts.com/learn/post/zxp-installer

---

### Step 2 — Install the Plugin

- Open the ZXP Installer  
- Drag & drop the `EaseControls.zxp` file into the installer  
- Follow the installation process  

---

### Step 3 — Launch After Effects

Open After Effects and find the panel under:

```
Window → Extensions → EaseControls
```

---

## Notes

- ZXP plugins require a separate installer because Adobe no longer supports direct installation.
- Make sure After Effects is closed during installation.
- If the panel does not appear, restart After Effects.
